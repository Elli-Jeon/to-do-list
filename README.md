### 1. commit 약속
#### 1.1 작성 방법
```
[행동] message (메시지는 명령조로 짧게)
```
#### 1.2   
|행동|용도|
|:---:|:---:|
|Add|새로운 파일 추가|
|Update|기존의 파일에 새로운 내용 추가|
|Fix|버그 수정 목적|
|Remove|기존 파일 삭제|
|Rename|기존 파일 이름 변경|
|DOC|md 같은 문서 작업|
|INIT|시작|
|Config|환경설정|
